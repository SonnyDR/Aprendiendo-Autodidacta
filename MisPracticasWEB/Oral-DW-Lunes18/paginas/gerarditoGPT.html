Algunos tipos de < inputs >

                    
        <input type="text" placeholder="Introduce tu nombre" />

un campo para entrada de texto. El control asociado a este campo es un cuadro de texto que permite a los usuarios editar una sola línea de texto sin formato. Los controles de texto son útiles para recopilar tiradas cortas de texto como títulos, nombres, apellidos, descripciones breves, palabras clave, etiquetas, etc.

                        
        <input type="password" placeholder="Introduce tu contrasenia" />

Como su nombre lo indica, una entrada con un tipo de contraseña crea una contraseña. Es automáticamente invisible para el usuario, a menos que sea manipulado por JavaScript.

                        
        <input type="email" placeholder="Introduce tu email" />

Cualquier entrada con el tipo de correo electrónico define un campo para ingresar una dirección de correo electrónico.

                        
        <input type="number" placeholder="Introduce numeros" />

Este tipo de entrada permite al usuario insertar solo números.

                        
        <input type="submit" value="Enviar" />

Utilice este tipo para agregar un botón de envío a los formularios. Cuando un usuario hace clic en él, envía automáticamente el formulario. Toma un atributo de valor, que define el texto que aparece dentro del botón.

                        
        <input type="search"

La entrada con el tipo de búsqueda define un campo de texto como un tipo de entrada de texto. Pero esta vez tiene el único propósito de buscar información. Se diferencia de escribir texto en que aparece un botón de cancelar una vez que el usuario comienza a escribir.

                        
        <input type="tel"

Un tipo de entrada de tel le permite recopilar números de teléfono de los usuarios.

Estilos propios de < input >

                    
        input  {
            caret-color : Color    
        }

Esta propiedad lo que nos permite es cambiar el color de la linea final de nuestro cursor al momento de que estemos escribiendo dentro de alguna caja de texto, ya sea un input, textarea, etc. En lugar de color predeterminado del sistema. Esta propiedad es especialmente util para la personalizacion de la interfaz de usuario y la mejora en elementos de formulario

                    
        input  {
            cursor : pointer    
        }

Este cursor muestra la mano esa que aparece cuando el contenido es un hiperenlace, mostrando al usuario que ese elemento se puede apretar, hacer click, para ir a otro lugar o desencadenar algún tipo de comportamiento

                    
        Textarea  {
            resize : none    
        }

En CSS existe una propiedad llamada resize ( redimensionar ), y los elementos textarea , por definición tienen resize: both;. O sea: podemos redimensionar tanto la altura como la anchura del elemento agarrando y estirando la esquina abajo derecha de este. Para que no se redimensione hay que poner el valor "none"

                    
        input  {
            -webkit-text-security : square    
        }

Esta propiedad se utiliza principalmente para aplicar un nivel de seguridad al texto, como ocultar caracteres sensibles como contraseñas o PIN en un campo de entrada de texto. Ya esta casi obsoleta, ya que no todos los navegadores admiten este estilo

Estilos globales para < Input >

                    
        input  {
            padding : 10px    
        }

Las paddingpropiedades CSS se utilizan para generar espacio alrededor del contenido de un elemento, dentro de los bordes definidos.

                    
        input  {
            border : 2px solid #fff    
        }

El estilo de CSS "border" se utiliza para agregar bordes a elementos HTML, como divs, imágenes, botones y otros elementos. Proporciona control sobre el diseño y la apariencia de estos elementos al definir cómo se verán los bordes que los rodean. El atributo "border" tiene tres partes principales que se pueden configurar: Grosor, Estilo del Border, Color del borde

                    
        input  {
            border-radius : 20px    
        }

La propiedad border-radius en CSS se utiliza para controlar la curvatura o redondez de las esquinas de un elemento HTML, como un div, un botón o una imagen.

                        
        input  {
            color : White    
        }

La propiedad CSS color se utiliza para establecer el color del texto dentro de un elemento HTML, como un < input > o cualquier otro elemento que contenga texto. Cuando se aplica a un < input >, generalmente se utiliza para definir el color del texto que se muestra dentro del campo de entrada.

                        
        input  {
            width : 200px    
        }

La propiedad CSS width se utiliza para establecer el ancho de un elemento HTML, incluido un elemento < input >. Cuando se aplica a un < input >, controla cuán ancho es el campo de entrada de texto en la página web. Esto puede ser útil para ajustar el diseño de tu formulario y controlar el espacio que ocupa un campo de entrada en particular.
